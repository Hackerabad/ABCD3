# 🎮 Phaser Game Development Session – Resources

This repository contains all the resources, templates, and materials used during the **Game Development Session** conducted using **Phaser.js**.

---

## About the Session

This session focused on helping students build a **simple playable browser game** using Phaser 3.

### What students learned:
- Basics of game development
- Phaser.js fundamentals
- Game loop (`preload → create → update`)
- Adding interactivity (click events)
- Score systems
- Sound & animations
- Using AI for game development (vibe coding)

## Learning Approach

We followed a simple philosophy:

> Build → Test → Improve

Students were encouraged to:
- Start small
- Use AI as a helper (not replacement)
- Iterate quickly

---

## Key Takeaways

- Game development is about logic + interaction
- Clean structure is more important than complexity
- AI can significantly speed up development
- A working game > a perfect idea
